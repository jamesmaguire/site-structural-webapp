Description

This is a website that solves simple structural engineering design problems. There is no warranty of correctness of these applications and results should be checked by by an experienced engineer.

Usage

Input boxes that look like are design inputs. Once an input is changed, the indicator on the top right of the page will show that the analysis must be re-run. To run the analysis press ENTER in any of the input fields.
