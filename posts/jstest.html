Explaination

This page is a test of setting up interactive elements (inputs/select/etc) using js, rather than HTML via org macros.

It will work by just providing an empty <div> or <span> with an id field and doing the rest in javascript.

Advantages include:

  • More flexible layout and inputs (can be customised easily without messy .org syntax)
  • Can spend more time working in js, rather than org file
